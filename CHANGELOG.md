# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-01-04

### Added
- Initial release of Flutter Performance Monitor
- Real-time FPS monitoring with color-coded indicators
- Frame time tracking in milliseconds
- Memory usage monitoring (platform-dependent)
- CPU usage estimation based on FPS
- Performance overlay widget with customizable positions
- Performance chart widget with history visualization
- Detailed performance dialog with statistics
- Tap to expand/collapse overlay functionality
- Long press for detailed performance information
- Comprehensive test coverage
- Example application demonstrating all features
- Full documentation and API reference
- Support for all Flutter platforms (iOS, Android, Web, Desktop)

### Features
- `PerformanceOverlay` widget for easy integration
- `PerformanceChart` widget for visual analytics
- `PerformanceTracker` for programmatic access
- `PerformanceMetrics` model with comprehensive data
- `PerformanceDialog` for detailed statistics
- Customizable appearance (colors, position, opacity)
- Configurable update intervals
- Statistics tracking (min, max, average FPS)
- FPS history with configurable retention
- Zero external dependencies

### Documentation
- Complete README with examples
- API reference documentation
- Best practices guide
- Troubleshooting section
- Platform support matrix
- Contributing guidelines

### Testing
- Unit tests for core functionality
- Widget tests for UI components
- Integration tests for full workflows
- 90%+ code coverage
